
# 💻 Cognifyz Front-End Development Project

## 📌 Project Overview
This project is an interactive and responsive frontend web application developed as part of the Cognifyz Technologies Front-End Development Internship.

The main objective of this project was to build a user-friendly interface while implementing core frontend development concepts such as responsive design, DOM manipulation, API integration, and form validation.

## 🚀 Features
- Responsive homepage design using Bootstrap  
- Interactive button with dynamic background color change  
- API integration to fetch and display user data  
- Contact form with input validation and success/error messages  
- Clean and user-friendly UI  

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript (Vanilla JS)
